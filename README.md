# Employee Knowledge Base RAG Chatbot

A learning project to build an Employee Knowledge Base RAG chatbot in n8n.

The chatbot will answer employee questions using company documents stored in Google Drive.

## Goal

Build a RAG chatbot from scratch using:

- n8n
- Google Drive
- Google Gemini
- Qdrant
- AI Agent
- Vector Store retrieval
- Chat memory

## Workflows

1. Document Indexing Workflow
2. Employee Chatbot Workflow
3. Error Handler Workflow

## Current Status

Project folder created. Architecture and setup planning in progress.
